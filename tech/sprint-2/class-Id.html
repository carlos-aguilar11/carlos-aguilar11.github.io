Class vs Id. Which one should you use?

Classes and IDs are both element selectors and their function are to identify an element based on the assigned parameters. The difference between them is that Class can be applied to a group of HTML elements on a single page and ID is applied only to one HTML element.

When you are starting your journey of learning to code, it's hard to identify when to use Classes vs IDs. I feel like there isn’t a wrong choice and you can easily use one or the other as they are very similar. So the way I wrap my mind around it is by thinking about which option is going to give me a better and clearer workspace. But, you could just think of what you want to achieve with this selector, changing one specific element, many, changing part of one, combining them together, the options are many.
